# C++ Chat Application

A real-time chat application built with C++ using socket programming.

## Features

- Multi-client support
- Real-time messaging
- Private messaging (/whisper)
- User list (/users)
- Join/leave notifications
- Cross-platform (Windows/Linux)

## Requirements

- C++11 compatible compiler
- POSIX threads (Linux) or Windows threads
- Network connectivity
